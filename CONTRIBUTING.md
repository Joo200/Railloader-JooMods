# Contributing

Thank you for interest in contributing to this project. We appreciate your help!

In order to keep the project healthy and compatible please take note of the following guidelines:

* The project has to be compatible with the latest version of Railroader.
* Please follow the code style and formatting guidelines provided in the project.
* Make sure to test your changes thoroughly before submitting a pull request.
* If you are adding new features, please provide documentation and examples.
* If you are fixing a bug, please provide a detailed description of the issue and how your changes fix it.
* If you are making significant changes, please open an issue first to discuss the changes with the maintainers.
* Keep the code simple and readable. Add comments for complex code and reasons for a specific implementation.
