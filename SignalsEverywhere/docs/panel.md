# CTC Panel

## Usage

* Use `N` to access the CTC Panel.
* Use `Right Click and Left Click` to throw knobs.
* Use `Right Click` on the Schematic to create or edit markers
* Use `Shift Click` in the signal box to throw knobs.

## Create new panels

You can create new panels by using the `panel.json` files as mixinto.

Look into the existing panel for more information.
